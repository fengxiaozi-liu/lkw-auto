# 洛克王国自动脚本

## 功能
- 自动完成每日任务
- 一键领取福利
- 可配置化任务流程

## 使用方法

### 1. 安装依赖
```bash
pip install pyautogui opencv-python
```

### 2. 配置分辨率
脚本基于 **2400x1080** 分辨率编写。
如果你的模拟器分辨率不同，需要修改 `config/game_config.py` 中的坐标。

### 3. 准备截图
将游戏截图放入 `images/` 目录：
- `main_menu.webp` - 主界面
- `quest_page.webp` - 任务界面

### 4. 运行
```bash
python scripts/lkw_bot.py
```

## 截图要求

### 主界面截图
需要包含：
- 任务按钮
- 福利按钮
- 返回按钮

### 任务界面截图
需要包含：
- 一键领取按钮
- 一键完成按钮

## 注意事项

1. **分辨率**：脚本坐标基于 2400x1080，如不一致请修改
2. **模拟器**：推荐使用雷电模拟器或夜神模拟器
3. **防检测**：脚本已添加随机延迟和坐标偏移
4. **安全**：建议低频使用，避免被封号
